Types of links:

Making hypertext uses the <a> tag. It has many attributes such as: "href", "name", "title", "target". The <a> tag is a inline as well as a block element, so allows the usage of both elements in side of it.

Internal Links:

These are links that are stored in the same folder as the current html file. The "href" attribute must be filled in with the .html file.
For example: internal link

External Links:

These links point to sites on different servers. The "href" attribute must be filled with the site URL. Setting "target" to "_blank" ensures that the link opens up in a new tab.
For Example: external link

Fragment Identifier Link:

These links point to sections in the current page itself. The "href" attribute must be filled with a hashtag followed by the section id or anchor name (eg. "#sectionID").
